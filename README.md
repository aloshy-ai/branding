## ASCII Art Usage

Run the ASCII art script:

```bash
curl -fsSL https://raw.githubusercontent.com/aloshy-ai/branding/main/ascii.sh | bash
```

Or 

```bash
curl -fsSL https://ascii.aloshy.ai | bash
```
